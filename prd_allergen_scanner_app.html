# Project: Allergen Scanner PWA (Swedish Food Database - Dabas) ## Overview A mobile-first Progressive Web App (PWA) that allows users to scan grocery barcodes (GTIN) and instantly check for allergens based on their personal profile. ## User Persona - Individuals with food allergies or intolerances. - Parents checking products for their children. - Health-conscious shoppers. ## Core Features - **Allergen Profile:** Users select allergens (Milk, Egg, Gluten, Nuts, Soy, etc.). - **Barcode Scanning:** Integration with device camera for GTIN scanning. - **Dabas API Integration (Simulated):** Fetching product names, ingredients, and allergen data. - **Color-Coded Alerts:** - **Green:** Safe to consume. - **Yellow:** May contain traces. - **Red:** Contains allergen. - **Accessibility:** High contrast, large fonts, and clear iconography. ## Screen Map 1. **Onboarding / Allergy Setup:** Initial screen to select allergens. 2. **Scanner Screen:** Minimalist UI with a large 'Scan' button and camera viewfinder. 3. **Product Detail - Safe:** Results screen showing product info with a prominent green "Safe" badge. 4. **Product Detail - Alert:** Results screen showing product info with a prominent red "Warning" and specific allergen highlighting. 5. **History/Saved:** (Optional/Future) List of recently scanned items. ## Technical Notes - Frontend: React-based PWA. - Scanning: ZXing or similar library integration. - Data: Local dummy data representing Dabas API responses for now.
